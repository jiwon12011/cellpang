# 세포팡 🍬 (Cellpang)

유미의 세포들 캐릭터로 즐기는 **힐링 매치-3 퍼즐** — 설치형 웹 앱(PWA).

> *"이기려는 매치-3가 아니라, 오늘의 내 감정을 돌보는 매치-3."*

[유미의 세포들 팬페이지](https://github.com/jiwon12011/Yumi-s-Cells) 프로젝트에서 분리한 독립 게임 앱입니다.

## ✨ 특징
- **애니팡 스타일 매치-3** — 인접 세포 스왑, 3매치+ 제거, 중력/리필, 캐스케이드 콤보
- **60초 타임어택** + 콤보 배수, 최고 점수 저장(localStorage)
- **핸드폰 앱처럼 (PWA)** — 홈화면 설치, 풀스크린 standalone, 노치/홈바 세이프에어리어, 오프라인 동작
- **관계도 세포 6종** 타일 (사랑/이성/감각/식욕/열정/응큼)

## 🎮 플레이
- 인접한 두 세포를 **드래그(스와이프)** 하거나 **차례로 탭** 해서 자리 변경
- 같은 세포 3개 이상 → 터짐 → 위 세포가 떨어져 채움 → 연쇄(콤보) 시 점수 배수 ↑

## 🚀 실행
PWA(설치·오프라인)는 `localhost` 또는 HTTPS에서 동작합니다(`file://` 불가).
```bash
# 저장소 루트에서
python3 -m http.server 8099
# → http://localhost:8099  접속
```
폰에서 같은 와이파이로 `http://<PC-IP>:8099` 접속 → 공유 → "홈 화면에 추가".

## 🗺️ 로드맵 (차별화)
- [ ] **프라임 세포 시스템** — 매 라운드 지배 감정이 점수 2배 (IP = 게임 규칙)
- [ ] **감정 분석 결과 리포트** — 가장 많이 터뜨린 세포로 "오늘의 마음" 분석 + 공유 카드
- [ ] **오늘의 감정세포(데일리)** + 출석 스트릭
- [ ] 감정 합체 콤보 / 불안세포 확산 / 응큼세포 히든 타일
- [ ] 유미 연애 서사 스토리 모드, 세포 도감

## 📁 구조
```
.
├─ index.html      # 화면 + HUD/오버레이 + SW 등록
├─ game.css        # 비주얼 (앱 셸, 폰 프레임, 세이프에어리어)
├─ game.js         # 매치-3 엔진
├─ sw.js           # 서비스워커 (오프라인 캐시)
├─ manifest.json   # PWA 매니페스트
├─ icons/          # 앱 아이콘 (192/512/180/32)
└─ assets/cells/   # 타일 세포 이미지 6종
```

## ⚙️ 설정 (game.js 상단)
`SIZE`(보드 8×8) · `ROUND_TIME`(60초) · `BASE_SCORE` · `TILES`
