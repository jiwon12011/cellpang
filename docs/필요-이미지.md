# 세포팡 — 이미지 제작 목록 (전체)

> 게임 룩: **통통한 젤리 캐주얼**(애니팡류 레퍼런스). 둥글둥글·쫀득, 두꺼운 **크림 테두리**, **글로시 하이라이트**(좌상단 빛), 부드러운 드롭섀도, 파스텔 + 선명한 포인트색.
> 공통 규격: **PNG 투명**(배경은 webp 가능), **2x 해상도**, 잘림 없이 중앙 정렬, 여백 포함.
> 파일명·경로 그대로 넣으면 코드가 자동으로 붙습니다. 표의 ✅는 **제가 임시 생성**해둔 것 — 더 예쁜 버전으로 덮어쓰면 됩니다.

조명·테두리·광택 규칙을 6 타일 기준으로 통일하면 전체 톤이 맞습니다.
